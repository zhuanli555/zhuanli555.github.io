c/c++调试工具gdb使用

发表于 2018-07-03 | 分类于 linux

gdb之前

gdb是gcc的调试工具，主要用于c/c++这两种语言的编程的调试。功能强大，主要体现在以下几点：

  • 可以按照用户自定义的要求随心所欲的运行程序
  • 可以设置断点，让程序在断点处停住
  • 当程序停住时，可以检查此时程序中运行状态
