leetcode 337. 打家劫舍 III

发表于 2018-06-26 | 分类于 leetcode

小偷又发现一个新的可行窃的地点。 这个地区只有一个入口，称为“根”。 除了根部之外，每栋房子有且只有一个父房子。 一番侦察之后，聪明的小偷意识到“这个地方的所有房屋形成了一棵二叉树”。 如果两个直接相连的房子在同一天晚上被打劫，房屋将自动报警。

在不触动警报的情况下，计算小偷一晚能盗取的最高金额。

示例 1:


     3
    / \
   2   3
    \   \
     3   1

能盗取的最高金额 = 3 + 3 + 1 = 7.

示例 2:


     3
    / \
   4   5
  / \   \
 1   3   1

能盗取的最高金额 = 4 + 5 = 9.
